1. This is a client-server File Transfer application in java.
  2. It is implemented using multithreading, hence is able to support multiple clients at once.
  3. It can be used to transfer files such as--
    • zip files
    • videos
    • audio files
    • documents
    • etc.
  4. This FTP can transfer GBs of data over Local Area Network or Hotspot.
  5. The app has an easy to understand interface at client side.
  6. to run:--
    • Clone the project and open it on Netbeans IDE.
    • First run server program Server.java
    • Then run Client.java
    • For sending messages/flies just select client from list of online clients
    • type in the message or choose the appropriate file and click on send msg/ send file respectively
